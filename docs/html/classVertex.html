Detailed Description

Class representing a vertex in the graph.

A vertex represents a node in the graph that can have outgoing and incoming edges, and can store various attributes such as location, ID, parking availability, and others. Vertices are used in graph traversal algorithms, and they store information about their adjacent vertices (edges).
